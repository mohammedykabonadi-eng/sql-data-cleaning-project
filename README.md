# Nashville Housing Data Cleaning

## Description
Data cleaning project using MySQL on the Nashville Housing dataset.

## Topics Covered
- Populating null values using self join
- Splitting address columns (street, city, state)
- Standardizing column values (Y/N → Yes/No)
- Removing duplicates using ROW_NUMBER()
- Dropping unused columns

## Tools
- MySQL 8.0
- DataGrip
